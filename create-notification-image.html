How to Create the Image:

  1. Option 1 - Download: Click the "Download as Image" button below to download a 512x512px PNG with transparent background and white "Top Hundred" text
  2. Option 2 - Online Tool: Use Canva or similar:
    • Create a 512x512px image
    • Set background to transparent
    • Add white text "Top Hundred" (bold, centered)
    • Export as PNG with transparency
  3. Option 3 - Image Editor: Use Photoshop, GIMP, or any image editor with transparent background
Note: Android will automatically add a colored circle (usually gold/amber) behind your transparent icon. The white text will show on top of this colored circle.
